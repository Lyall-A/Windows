VB-CABLE C+D (Audio Device driver)
-----------------------------------------------------------------

Programmed by V.Burel / JS-Loezic

Copyrights V.Burel©2010-2019 All rights reserved.

VB-CABLE is an Audio Driver working as virtual audio cable.
All signals sent to the device output is going on the device input.

VB-CABLE Works on Windows XP, Vista, Win7, 8.1, 10, Server 2003,
Server 2008, server 2012, server 2016 (32 or 64bits).

To support our effort and development team, it is possible to make 
donation on www.VB-CABLE.com web site. Thanks by advance for your 
participation.


-----------------------------------------------------------------
INSTALLATION
-----------------------------------------------------------------

Just UNZIP the package and run VBCableCD_Setup program

REBOOT AFTER INSTALLATION.
(REBOOT AFTER DE-INSTALLATION)
